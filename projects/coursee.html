Coursee

Ruby on Rails, HTML/CSS, Javascript, Leaflet

Coursee is a simple application that keeps track all of your running courses and share it with other users. This project is designed using the Ruby on Rails framework and leaflet plugin.

Some of the features in Coursee:

  • Draw running courses on map
  • Save a screenshot of the courses path on map
  • Post your running course and share it with other users
  • See other user's running courses
