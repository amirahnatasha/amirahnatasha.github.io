Classic Space Invaders in Love2D

Lua, Love2D framework, 2D Game Development

This is a remake of the classic space invaders game which was first released in 1978 by Taito, Japan . This project is fully written in Lua programming language and was developed using Lua's Love2D framework. Love2D framework is a free, open-source framework used in developing 2D video games.

How to play:

Simply move the player using the left and right arrow keys. Press the spacebar to shoot bullets. Game will be over if the alien reaches the player or if the player is hit by the alien bullets.
